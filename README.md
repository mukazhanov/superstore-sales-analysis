# 🛒 Superstore Sales Analysis

**Бизнес-вопрос:** Какие категории товаров приносят убыток и почему?

---

## 📊 О проекте

Анализ 9 994 заказов американского ритейлера за 2014–2017 гг.  
Цель — найти убыточные сегменты и дать конкретные рекомендации бизнесу.

**Стек:** Python · pandas · matplotlib · seaborn · Jupyter Notebook

---

## 🔍 Ключевые находки

| Находка | Цифры |
|---------|-------|
| **Tables** приносят убыток | −$17 725 при продажах $207K (маржа −8.6%) |
| **Bookcases** в убытке | −$3 473 при продажах $115K (маржа −3.0%) |
| **Скидки >20%** уничтожают прибыль | Средняя прибыль: −$45 при 21–30% скидке |
| **Furniture** — слабейшая категория | 41% продаж, но лишь 6% общей прибыли |
| **Technology** — лидер | 17.4% маржа против 2.5% у мебели |

---

## 💡 Рекомендации

1. **Ограничить скидки выше 20%** — ввести согласование с менеджером
2. **Пересмотреть ценообразование на Tables** или выйти из сегмента
3. **Масштабировать Copiers** (маржа 37.2% — лучший показатель в портфеле)
4. **Аудит Central-региона** — там убытки концентрируются сильнее всего
